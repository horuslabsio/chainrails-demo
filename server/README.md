# Server Implementation

This folder contains sample server implementations of the Chainrails API in different languages.

For each Language, you'd find:
- A detailed, commented implementation for supported features
- A sample application integrating Chainrails
- A CLI interactive playground to quickly get a hang of how Chainrails work

**PS: This sample implementations can only become better if you contribute to it. We're accepting:**
- Bug fixes and improvements
- Implementation in other languages
- Typographical corrections, suggestions etc.
